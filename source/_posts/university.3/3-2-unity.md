## 教程

似乎是有14個，想過一遍大概也很快。

###  1.Interactive Tutorials
1. Play & Edit Mode
  學會進入遊戲和編輯模式。遊戲模式中對場景做的任何修改都是不會保留的。
2. Game Objects & Components
  教我們如何給遊戲物體添加剛體組件來添加物理行為。
3. Tweaking Components
  教我們調整組件的值來使得行為符合預期。
4. Prefabs Power
  教我們使用預設。
